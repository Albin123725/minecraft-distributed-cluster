# Minecraft Distributed Cluster

A fully distributed Minecraft cluster with 16 game servers, custom world, plugins, proxy, and management dashboard.

## Features
- 16 Game Servers (6.47GB Total RAM)
- Your Custom World + Plugins
- BungeeCord Proxy Load Balancing
- Web Management Dashboard
- Auto-scaling Architecture

## Connection
- **Proxy:** `mc-proxy-main.onrender.com:25575`
- **Dashboard:** `https://mc-management.onrender.com`

## Total Capacity
- 400 Players
- 16 Different Regions
- 6.47GB Cluster RAM
- Your Custom World & Plugins
